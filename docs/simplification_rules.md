```
# Identity and zero elements
0 * x = 0
0 + x = x
1 * x = x
x / x = 1

# Simplify negations
(-(-x)) = x
(-x) + (-y) = -(x + y)
(-x) * (-y) = x*y

# Fractions
(a / b) * (c / d) = (ab)/(cd)
(a / b) / (c / d) = (a / b) * (d / c)

# Expansion
a * (b + x) = ab + ac
exp(x + y) = exp(x) * exp(y)
a.pow(x + y) = a.pow(x) * b.pow(y)

# Trig identities
pow(sin(x), 2) + pow(sin(y), 2) = 1

# Equivalent expressions
a + (b + c) = (a + b) + c
a * (b * c) = (a * b) * c
```
