# Syntax

**Mapping string keys to numerics by assignment operator**
```
>> x = 5
x = 5
>> some_string = 2 + x
some_string = 7
>> x * x
ans = 49
```

**Some string keys are reserved**
```
>> exp = 5
Error XXX: exp is a reserved keyword
```
